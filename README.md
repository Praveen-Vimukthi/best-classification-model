# Best Model Predictor

A machine learning web application that automatically identifies the best-performing model for a given dataset.

## Overview

Selecting an appropriate machine learning model is a critical step in any data-driven project. This application simplifies that process by training multiple models and evaluating their performance to determine the most accurate one.

## Features

- Upload datasets in a simple interface  
- Automatic data preprocessing  
- Train multiple machine learning models  
- Evaluate and compare model performance  
- Automatically select the best model  
- Clean and user-friendly interface  

## Supported Models

- Logistic Regression  
- Decision Tree  
- Random Forest  
- Support Vector Machine (SVM)  
- K-Nearest Neighbors (KNN)  

## Tech Stack

Frontend:
- React (if applicable)

Backend:
- Python (Flask)

Machine Learning:
- Scikit-learn  
- Pandas  
- NumPy  

## How It Works

1. Upload a dataset  
2. Configure parameters such as test size  
3. The system trains multiple models  
4. Each model is evaluated using accuracy  
5. The best-performing model is selected and returned
